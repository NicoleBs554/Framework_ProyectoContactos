# 📇 Contact Keeper – Aplicación de Contactos con React + Vite

Aplicación web moderna para la gestión de contactos personales. Permite **crear, leer, actualizar y eliminar** contactos (CRUD completo) con un sistema de autenticación básica, múltiples vistas adaptables, popup de detalles y persistencia en `localStorage`. Desarrollada con **React 18**, **Vite** y **React Router DOM**.

🔗 **Repositorio:** [NicoleBs554/Framework_ProyectoContactos](https://github.com/NicoleBs554/Framework_ProyectoContactos.git)

---

## 🚀 Tecnologías principales

- **React 18** – Biblioteca de UI
- **Vite** – Bundler ultrarrápido
- **React Router DOM** – Enrutamiento y protección de rutas
- **Context API** – Gestión de estado global (autenticación y contactos)
- **uuid** – Generación de IDs únicos para cada contacto
- **CSS moderno** – Diseño responsive, modo oscuro de alto contraste y crop de imágenes

---

## 📋 Requisitos previos

- Node.js (versión 18 o superior) y npm instalados
- Git (opcional, para clonar el repositorio)

---

## 🔧 Instalación y puesta en marcha

Sigue estos pasos en tu terminal (Linux, macOS o WSL en Windows):

```bash
# 1. Clonar el repositorio
git clone https://github.com/NicoleBs554/Framework_ProyectoContactos.git

# 2. Acceder a la carpeta del proyecto
cd Framework_ProyectoContactos

# 3. Instalar dependencias
npm install

# 4. Iniciar el servidor de desarrollo
npm run dev
